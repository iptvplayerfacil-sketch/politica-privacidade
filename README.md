# politica-privacidade
Política de Privacidade do aplicativo IPTV Player

Última atualização: 14/01/2026

Este aplicativo respeita a sua privacidade e está comprometido em proteger os dados pessoais dos usuários.

## Coleta de Informações
O aplicativo não coleta, armazena ou compartilha informações pessoais identificáveis.

## Uso de Informações
Nenhuma informação pessoal é utilizada para qualquer finalidade.

## Compartilhamento de Dados
Não compartilhamos dados com terceiros.

## Conteúdo de Terceiros
O aplicativo pode exibir conteúdos fornecidos por terceiros, os quais são de responsabilidade de seus respectivos provedores.

## Segurança
Adotamos medidas adequadas para proteger o aplicativo contra acesso não autorizado.

## Alterações nesta Política
Esta política pode ser atualizada a qualquer momento sem aviso prévio.
